# HIGRAORFA

**Sistema de Gerenciamento para Academias**

Desenvolvido por **Rafael Alcântara**, **Higor dos Santos** e **Paulo Roberto**.

---

##  Sobre o Projeto

O **HIGRAORFA** é um sistema de gerenciamento para academias desenvolvido com o objetivo de modernizar e simplificar processos administrativos, proporcionando maior controle sobre alunos, planos, mensalidades e operações internas.

A plataforma foi projetada para atender academias de pequeno e médio porte, oferecendo uma solução intuitiva, eficiente e escalável.

---

##  Objetivos

* Digitalizar processos administrativos.
* Reduzir o uso de controles manuais.
* Melhorar a organização da academia.
* Facilitar o acompanhamento financeiro.
* Garantir uma gestão mais eficiente dos alunos e planos.

---

## Funcionalidades

###  Gestão de Alunos

* Cadastro de alunos.
* Atualização de dados cadastrais.
* Consulta de informações.
* Exclusão de registros.

###  Gestão de Planos

* Cadastro de planos.
* Alteração de valores e benefícios.
* Associação de planos aos alunos.

###  Controle de Mensalidades

* Registro de pagamentos.
* Consulta de mensalidades.
* Controle de inadimplência.
* Histórico financeiro.

###  Administração

* Gerenciamento geral da academia.
* Organização de dados e registros.
* Relatórios administrativos.

---

##  Arquitetura e Desenvolvimento

O HIGRAORFA foi desenvolvido seguindo boas práticas de engenharia de software, utilizando uma arquitetura organizada em camadas para garantir maior manutenção, escalabilidade e organização do código. O sistema implementa o padrão **DAO (Data Access Object)** para o gerenciamento do acesso aos dados, o padrão **Singleton** para controlar uma única instância da conexão com o banco de dados e operações **CRUD completas** para cadastro, consulta, atualização e exclusão de informações. A separação entre entidades, regras de negócio e persistência de dados torna a aplicação mais modular, reutilizável e de fácil manutenção.
