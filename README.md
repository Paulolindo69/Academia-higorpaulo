# HIGRAORFA

Sistema de gerenciamento para academias  
Desenvolvido por Rafael Alcântara e Higor dos Santos

---

## Sobre o Projeto

O HIGRAORFA é um sistema criado para facilitar a administração de academias, oferecendo controle simples e eficiente de alunos, planos e pagamentos.

O objetivo é tornar a gestão mais organizada, prática e digital.

---

## Funcionalidades Principais

- Cadastro de alunos  
- Controle de mensalidades  
- Gerenciamento de planos  
- Organização administrativa  

---

## Objetivo

Proporcionar uma solução intuitiva e funcional para academias de pequeno e médio porte, otimizando tempo e reduzindo processos manuais.
